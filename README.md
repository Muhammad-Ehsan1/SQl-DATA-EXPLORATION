# SQl-DATA-EXPLORATION-
Conducted an in-depth data exploration project on COVID-19 deaths and vaccinations using Microsoft SQL Server. The analysis involved examining total cases, deaths, and infection rates by country and continent, as well as calculating death percentages relative to cases and population. Additionally, the project analyzed the global vaccination progress and the relationship between population and vaccinations.
